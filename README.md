GHP(General Hypertext Preprocessor)通用型超文本预处理器(曾用名SEHP超级拓展超文本预处理器)是一个通用的、高性能的、高并发的、安全的超文本预处理器，旨在让程序员不用学习新的编程语言，可直接将C/C++、Java、Python、Go、JS、易语言等主流编程语言直接嵌入HTML,无需进行新语法的学习,直接上手编写网站前端和后端。
通用型超文本预处理器是一个系统，由超文本预处理引擎、Web Server、WebAssembly预处理器以及DOM Plus组成。兼顾了Web前端及后端开发。
项目整体是为了解决当前在网站编程时的学习门槛高、技术繁杂、运行环境复杂等的问题。项目整体顺应编程普遍化、简单化的理念。
